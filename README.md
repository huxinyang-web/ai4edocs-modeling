# Ai4Energy Docs - Modeling & Simulation

Ai4Energy 文档 - 建模与仿真

## 内容

- **建模**: MTK 建模、物理系统建模、微分方程
- **仿真**: 数值仿真方法、仿真案例分析、性能优化

## 部署

本项目使用 VitePress 构建，部署在 GitHub Pages：
https://huxinyang-web.github.io/ai4edocs-modeling/

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
